# Demand-Skills-for-Data-Scientists
This project is a part of the subject Problem Solving in Information Technology (PSIT), Faculty of Information Technology, King Mongkut's Institute of Technology Ladkrabang (KMITL).

This project is about doing data analysis. We've chosen the title Demand Skills for Data Scientists for this project. Our main target is to how should data scientists who want to be in demand by employers spend their learning budget?

เนื่องจากในปัจจุบันเป็นยุคของข้อมูล "Big Data" หรือ "ข้อมูลขนาดใหญ่" ได้กลายเป็นทรัพยากรที่มีค่า เพราะเป็นอีกหนึ่งกลไกสำคัญขององค์กร
ทั้งภาคธุรกิจและรัฐบาลในยุคปัจจุบัน เนื่องจาก ข้อมูลขนาดใหญ่ เป็นสิ่งที่รวบรวมพฤติกรรม รสนิยม รวมถึงความคิดเห็นของผู้คนบนโลกต่อสิ่งรอบ
ตัวต่างๆ แทบทุกกิจกรรมที่เราทำในหนึ่งวัน กลายเป็นข้อมูลที่นำไปใช้ต่อได้ ไม่ว่าจะเป็นการต่อยอดทางธุรกิจเพื่อสร้างสินค้าและบริการให้เข้าถึงกลุ่มผู้
บริโภคเป้าหมายได้มากขึ้น หรือนำไปใช้เพื่อออกแบบนโยบายภาครัฐให้เข้ากับความต้องการของประชาชน
อย่างไรก็ตาม ข้อมูลจำนวนมหาศาลนี้ ไม่สามารถนำมาปรับใช้ได้ในทันที ด้วยขนาด ความเร็ว และความหลากหลายของข้อมูล 
ทำให้องค์กรต้องการคนที่สามารถทำความเข้าใจข้อมูลและดึงแก่นสำคัญออกมาใช้ได้ ซึ่ง "Data Scientist" คือ ผู้ที่จะเข้ามาจัดการ
กับข้อมูลเหล่านี้ ซึ่งทำหน้าที่วิเคราะห์ ศึกษาข้อมูลเป็นหลัก
	ปัจจุบัน Data Scientist เริ่มเป็นที่รู้จักและสนใจของผู้คน คนที่อยากจะเป็น Data Scientist เริ่มมีมากขึ้น ถึงอย่าง
นั้นก็ยังไม่เพียงพอต่อความต้องการของตลาด การ Analysis ครั้งนี้ จัดขึ้นเพื่อให้ผู้ที่สนใจ หรือ เป็น Data Scientist ที่อยากหาความรู้
เพิ่มเติมเพื่อพัฒนาความสามารถของตัวเอง ได้ตัดสินใจง่ายขึ้นว่าจะศึกษา ภาษาใด หรือ ต้องมีทักษะทางด้านใด เพื่อจะสามารถพัฒนาตนเองในสายงานได้อย่าง
